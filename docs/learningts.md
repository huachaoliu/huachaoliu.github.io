## 基础
* [原始数据类型]()
    * [原始数据类型]()
        ### 布尔值
        ```ts
        {
            let isDone: boolean = false;
        }
        ```
        编译结果:
        ```js
        {
            var isDone = false;
        }
        ```
        ##### ps:使用构造函数创建的对象不是布尔值;
        ### 数值
        ```ts 
        {
            let age: number = 25;
        }
        ```
         编译结果:
        ```js
        {
            var age = 25;
        }
        ```
        ### 字符串
        ```ts
        {
            let name: string = 'misliu';
        }
        ```
         编译结果:
        ```js
        {
            var name = 'misliu';
        }
        ```
